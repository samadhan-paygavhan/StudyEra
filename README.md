# Study Era - Full Stack MERN Application

Study Era is a comprehensive educational platform built using the MERN stack. It features a robust authentication system with email verification and a clean, responsive UI.

The project follows a **Mono-repo** structure to manage both the frontend and backend efficiently.

- **Frontend:** React.js (Vite)
- **Backend:** Node.js & Express.js (MVC Pattern)
- **Database:** MongoDB
- **Authentication:** JWT with Email Verification (Nodemailer)
